## 2. KPI Methodology Alignment
The following definitions from the `aso_metric_glossary.md` are formally accepted as the Single Source of Truth (SSOT):
* **Net Revenue:** Must exclude all duplicate transactions flagged by the 10-second deduplication logic.
* **Peak Hours:** Exclusively defined as the top 12% of revenue-generating hours (~20 hours/week).
* **Redownload Ratio:** Target benchmark is set at 2.0 (66.7% redownloads).
